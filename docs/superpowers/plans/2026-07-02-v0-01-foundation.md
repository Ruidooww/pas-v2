# V0-01 Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the pas-v2 V0 engineering foundation so backend, frontend, Docker, and provider-neutral CI commands are runnable from the new repository.

**Architecture:** Use a pnpm monorepo with `apps/backend` for the NestJS API and `apps/frontend` for the Vite React SPA. Keep deployment to four PAS-owned services: `pas-frontend`, `pas-backend`, `pas-postgres`, and `pas-redis`; RAGFlow remains external.

**Tech Stack:** Node.js 22, pnpm 11, TypeScript 5, NestJS 11, Vite 7, React 19, Ant Design 5, Vitest, Docker Buildx, Nginx.
